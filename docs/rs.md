### 钟铭（天都镇）

```

字号：字子任
年龄：32
性别：男
身份/职业：天策府偏将
身高：187

喜/恶：喜坦荡，恶捕风捉影
武器：长枪，弓
备注：有一来自西域的棕鬓良驹，名唤赤风
```
